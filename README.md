# number_counter
Adder of numbers written with Python, has a version that also has a translator
when you open it asks you, for number, and then takes your numbers, and then number + number, and then print the result
## availible languages
english and persian
### required library
googletrans
I just learned this library and I decided to use it in this small project and work with it little by little, so it is not like the previous projects I made, and the Persian version is also available in the folder separately.
